# Tarot Arcana
หน้าเว็บสำหรับเปิดไพ่ทาโรต์ที่ใช้งานง่ายบนเดสก์ท็อปและอุปกรณ์พกพา

- รองรับการอ่านไพ่หลายรูปแบบทั้งแบบทั่วไปและแบบแยกตามหัวข้อ
- แสดงคำทำนายรายใบพร้อมสรุปภาพรวมของการเปิดไพ่ในแต่ละรอบ
- คงโครงสร้างไฟล์ให้เรียบง่าย เพื่อให้แก้ไขข้อความ รูปภาพ และรูปแบบการอ่านไพ่ได้สะดวก

## ฟีเจอร์หลัก

- เปิดไพ่แบบ 1 ใบ, 3 ใบ, 5 ใบ และ 10 ใบ
- มีหมวดการเปิดไพ่แยกตามหัวข้อ ได้แก่ ทั่วไป ความรัก สุขภาพ การงาน และการเงิน
- รองรับไพ่กลับหัว
- รองรับขั้นตอนสับไพ่และตัดไพ่แบบเลือกใช้ได้ ไม่บังคับ
- แสดงคำทำนายรายใบพร้อมรายละเอียดแยกด้านงาน ความรัก และการเงิน
- แสดงคำสรุปภาพรวมจากไพ่ที่เปิดได้จริงในรอบนั้น
- ใช้ภาพไพ่จากโฟลเดอร์ `image/`
- ทำงานได้โดยไม่ต้องใช้ฐานข้อมูลหรือบริการภายนอก

## โครงสร้างไฟล์

- `index.html` โครงสร้างหน้าเว็บและองค์ประกอบหลักของส่วนติดต่อผู้ใช้
- `style.css` รูปแบบการแสดงผล สี ฟอนต์ การจัดวาง และอนิเมชัน
- `script.js` ตรรกะการทำงานของระบบเปิดไพ่ การสุ่มไพ่ และการสร้างคำทำนาย
- `image/` ไฟล์ภาพหน้าไพ่และภาพหลังไพ่

## วิธีใช้งาน

1. เปิดไฟล์ `index.html` ผ่านเว็บเบราว์เซอร์
2. ระบุคำถามหรือหัวข้อที่ต้องการดูได้ตามต้องการ
3. เลือกรูปแบบการเปิดไพ่จากรายการที่ระบบเตรียมไว้
4. เลือกว่าจะเปิดใช้ไพ่กลับหัวหรือไม่
5. หากต้องการ สามารถกดสับไพ่หรือตัดไพ่ก่อนเปิดไพ่ได้
6. กดเปิดไพ่เพื่อดูผลการอ่านและคำสรุปภาพรวม

## แนวทางการปรับแต่ง

หากต้องการปรับข้อความหรือรูปแบบการอ่านไพ่เพิ่มเติม สามารถแก้ไขได้จากไฟล์ดังนี้

- เพิ่มหรือแก้ spread ใน `script.js`
- ปรับข้อความแนะนำและคำอธิบายบนหน้าเว็บใน `index.html`
- เปลี่ยนสี ขนาด ระยะห่าง และลักษณะการแสดงผลใน `style.css`
- เปลี่ยนภาพไพ่หรือภาพหลังไพ่ในโฟลเดอร์ `image/`
